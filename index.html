AI Framework — Igris

The idea for this framework came from a conversation with a friend: “What if we could generate automated scripts from simple words?” From that spark, I built Igris, an AI‑powered testing framework that connects with OpenAI to generate a JSON plan that functions as test cases, using SauceDemo as the demo site. Over time, and after getting feedback, I’ve added new features and integrated CI/CD through GitHub Actions and workflow automation.

While designing it, I followed an interpreter pattern so the tool can adapt to different teams and projects depending on their needs. Right now, it uses Pytest/Playwright, but by updating or creating a JSON plan translator, it can be extended to other stacks such as Java/Selenium or Cypress in JavaScript/TypeScript.

API Framework — GoRest

An API test automation framework built against the GoRest public REST API using Postman collections, Newman, and GitHub Actions. Authentication validation, a full data-driven CRUD lifecycle, and negative contract testing — all wired into a CI/CD pipeline.

While working on the CI/CD part, I ran into an unexpected issue — Cloudflare was blocking the requests, which led me to rethink my approach and how to show the pipeline was working.

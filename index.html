AI Framework — Igris

Igris generates, executes, and reports UI tests from plain text instructions. Describe a scenario in English — the AI produces a structured JSON test plan, Playwright runs it, and Allure captures every step with screenshots. No scripting required.

The interpreter pattern keeps the AI output portable: the same JSON plan that runs today in Python/Playwright can be picked up by a TypeScript, Java, or Selenium translator without touching the AI layer.

API Framework — GoRest

An API test automation framework built against the GoRest public REST API using Postman collections and Newman. Tests run from the command line or automatically through GitHub Actions on every push. The pipeline handles a real-world infrastructure challenge: GoRest sits behind Cloudflare, which blocks cloud server IPs — a dedicated gate collection detects this gracefully and controls whether the remaining jobs run.

Collection 01 — CI Environment Check

Acts as a pipeline gate. The test expects a 403 Cloudflare block — if blocked, the pipeline skips the API tests gracefully. If the API is actually accessible, the gate opens and the remaining collections run.

Collection 02 — Authentication Validation

Validates both sides of auth: an invalid token returns 401, a valid token returns 200. Each request asserts status code, response time under 2000ms, valid JSON, and correct Content-Type.

Collection 03 — End-to-End Flow

Full CRUD lifecycle: create user → post → comment → delete all, run across 4 data iterations from a CSV file. Resource IDs are chained across requests. Response time assertions run at collection level to avoid duplication.

Collection 04 — Negative Validation

Tests API contract boundaries: missing fields, invalid gender, invalid email, duplicate email, and empty body — all expecting 422 with the correct error target. Duplicate email uses dynamic pre-request data generation to avoid test pollution.

GitHub Actions — Full Pipeline Flow

The pipeline runs on every push and PR to main. Four jobs run in parallel after the gate check passes, each uploading its Newman report as a downloadable artifact.
